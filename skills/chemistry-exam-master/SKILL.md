### 三、三大压轴题解题模型

#### 模型1：无机推断题三步法
```
突破口：颜色/状态/特征反应/特定条件/数据
    ↓
假设验证：代入各选项验证推断→多重约束排除
    ↓
逻辑闭环：正向验证所有反应合理+逆向验证所有现象匹配
```

**常见突破口速查：**
| 特征 | 可能物质 |
|:----|:---------|
| 淡黄色固体 | S、Na₂O₂、AgBr |
| 红褐色沉淀 | Fe(OH)₃ |
| 使品红褪色 | SO₂（加热恢复）、Cl₂（不恢复） |
| 三角转化 | Al→Al₂O₃→Al(OH)₃→NaAlO₂；Fe(II)→Fe(III)→Fe 等 |
| 连续氧化 | H₂S→S→SO₂→SO₃；NH₃→NO→NO₂→HNO₃ |

#### 模型2：工艺流程题四段法
```
原料 → 预处理(粉碎/浸出/焙烧/酸溶)
     → 核心反应(氧化还原/酸碱中和/沉淀转化)
     → 分离(过滤/萃取/蒸馏)
     → 产品(蒸发结晶/重结晶/电解)

每个环节追问：①为什么这样做？②如何操作？③能否改进？
```

#### 模型3：有机推断逆推法
```
目标产物 → 逆推前一步原料 → 再逆推 → 直到已知起始物
    ↓
正向书写完整路线，每个箭头标注反应条件（试剂/温度/催化剂）
```

### 四、常见失分陷阱（80+必须避开）

| 陷阱 | 表现 | 应对 |
|:----|:-----|:-----|
| NA陷阱 | 标准状况下非气态（H₂O/CHCl₃/SO₃等用22.4L） | 检查是否为气态 |
| 离子方程式 | 拆分错误（弱电解质/沉淀/气体不拆） | 记牢强弱电解质清单 |
| 氧化还原顺序 | Fe与Cl₂→FeCl₃（非FeCl₂） | Fe作为还原产物由氧化剂强弱决定 |
| 平衡移动 | 勒夏特列原理被条件变化误导 | 先判断是浓度/压强/温度变化 |
| 电化学 | 原电池正负极与电解池阴阳极混淆 | 原电池→自发氧化还原→负极氧化 |
| 有机条件 | 消去→浓硫酸170℃；酯化→浓硫酸加热 | 逐一记住每类反应的条件差异 |
| 盐类水解 | 忘记弱离子水解（Na₂CO₃→碱性） | 记"谁弱谁水解" |

### 类型二：知识体系梳理 → 模块化输出

```
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
🧪 化学反应原理 — 80+必过
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

├─ 反应热
│   ├─ ΔH=生成物总焓-反应物总焓
│   ├─ 盖斯定律：分步反应热之和=总反应热
│   ├─ 计算：已知键能/燃烧热求ΔH
│   └─ 易错：ΔH符号与放热/吸热对应
├─ 化学反应速率
│   ├─ v=Δc/Δt，影响条件(浓度/温度/压强/催化剂)
│   ├─ 速率方程/活化能/催化剂机理
│   └─ 易错：压强变化只影响气体反应
├─ 化学平衡
│   ├─ 勒夏特列原理：改变条件→平衡向削弱该改变的方向移动
│   ├─ 平衡常数K=c(C)·c(D)/c(A)·c(B) — K只受温度影响
│   ├─ 转化率计算：三段式法（起始→变化→平衡）
│   └─ 易错：催化剂不改变平衡，只加快到达平衡
├─ 水溶液离子平衡
│   ├─ 弱电解质的电离平衡（电离度/Ka/Kb）
│   ├─ 水的离子积Kw=c(H⁺)·c(OH⁻)=10⁻¹⁴(25℃)
│   ├─ 盐类水解（谁弱谁水解，谁强显谁性）
│   └─ 易错：HCO₃⁻既能电离又能水解
├─ 电化学
│   ├─ 原电池：负极氧化，正极还原
│   ├─ 电解池：阳极氧化，阴极还原
│   └─ 易错：放电顺序要分清（阳极活性电极优先溶解）
└─ 80+秘笈：原理题先定性后定量，图像题先标坐标再读曲线
```

## 注意事项

1. **化学用语规范** — 所有方程式使用规范的化学式/离子符号（非中文描述）
2. **条件不可省略** — 每个反应标注条件（高温/点燃/催化剂/光照等）
3. **配平三查** — 原子守恒、电荷守恒（离子方程式）、电子守恒（氧化还原）
4. **有机物规范** — 结构简式中C-H键省略，官能团完整显示
5. **表述规范** — 用规范化学语言（非日常语言）描述现象和过程
6. **数字精确** — 计算题保留有效数字，标注单位
7. **80+标准** — 不涉及超纲内容，侧重易错点和提分技巧
8. **推断逻辑** — 推断题要求展示完整的推导过程而非仅给答案
