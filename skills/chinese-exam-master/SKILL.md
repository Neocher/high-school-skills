### 三、现代文阅读答题模板

#### 模板1：论述类文本阅读

```
【三看法】
一看：标题和出处（判断文章类型和立场）
二看：首尾段（把握中心论点）
三看：每段首尾句（提炼分论点）

【选择题解题步骤】
1. 定位：找到选项在原文中的对应位置
2. 比对：逐字比对，尤其注意以下易错类型：
   ├─ 偷换概念（主语/宾语被替换）
   ├─ 范围不当（全部→部分，可能→一定）
   ├─ 因果错乱（强加因果/因果倒置）
   ├─ 无中生有（原文未提及）
   └─ 曲解文意（与原文表述不一致）
3. 排除：用排除法缩小范围
```

#### 模板2：文学类文本阅读（小说）

```
【四步分析法】
1. 情节分析：梳理故事脉络，找开端→发展→高潮→结局
2. 人物分析：外貌/语言/动作/心理/细节描写，概括性格特征
3. 环境分析：自然环境（烘托气氛/暗示命运）+社会环境（交代背景）
4. 主题分析：通过人物命运和故事情节归纳主题

【常见题型答题模板】
▎作用题："XXX在文中有何作用？"
答：①内容上…… ②结构上…… ③主题上……

▎赏析题："赏析画线句子"
答：①手法（修辞/描写/表现手法） ②内容（写了什么） ③效果（生动形象地写出了……）
```

#### 模板3：实用类文本阅读

```
【信息筛选整合】
1. 快速浏览，圈画关键词（时间/数据/观点/结论）
2. 根据题目定位原文
3. 整合答案：不照抄原文，用自己的话概括

【图表题】
1. 读图：标题→坐标轴→数据趋势→极值点
2. 转述：将图表信息转化为文字表述
3. 结论：图表反映了什么趋势/现象
```

### 四、古诗词鉴赏框架

```
【四步鉴赏法】

1️⃣ 看题目（时间/地点/题材/体裁）
  ├─ 送别诗：不舍/勉励/思念
  ├─ 边塞诗：报国/思乡/苦寒
  ├─ 山水田园诗：热爱自然/归隐/闲适
  ├─ 咏史怀古诗：借古讽今/昔盛今衰
  └─ 羁旅思乡诗：孤独/思亲/漂泊

2️⃣ 看意象（典型意象的象征意义）
  ├─ 月：思乡  │ 柳：送别
  ├─ 雁：思归  │ 菊：隐逸
  ├─ 流水：时光 │ 落花：春去/无常
  └─ 酒：愁绪  │ 笛声：思乡

3️⃣ 看手法（表现手法/修辞手法）
  ├─ 抒情方式：直抒胸臆/借景抒情/托物言志
  ├─ 表现手法：虚实结合/动静结合/对比/用典
  ├─ 修辞手法：比喻/拟人/夸张/对偶/借代
  └─ 结构技巧：铺垫/照应/卒章显志

4️⃣ 看情感（结合注释和作者背景）
  ├─ 格式答案：通过……（意象/手法），表达了……（情感）
  └─ 答题模板：①手法 ②内容 ③情感

⚠️ 常见失分：只写术语不结合内容分析
```

### 五、议论文写作模板（80+核心能力）

#### 文章结构：六段式

```
第1段：引论（100字）
├─ 引材料/名言/现象
└─ 亮观点（中心论点，一句话明确）

第2段：分论点一（150字）
├─ 分论点句（段首）
├─ 论据（事实/数据/名人事例）
└─ 分析论证

第3段：分论点二（150字）
├─ 分论点句
├─ 论据
└─ 分析论证

第4段：分论点三（150字）
├─ 分论点句（递进或转折）
├─ 论据
└─ 分析论证

第5段：思辨深化（100字）
├─ 辩证看待（承认反面观点合理性）
└─ 更高层次总结（跳出非黑即白）

第6段：结论（100字）
├─ 重申观点（呼应开头但换表述）
├─ 升华（联系时代/青年责任）
└─ 号召或展望
```

### 六、语言文字运用核心题型

#### 题型1：成语题
```
解题步骤：
1. 看语境（褒贬/对象/程度）
2. 看搭配（能否修饰后面成分）
3. 常见陷阱：望文生义/对象误用/褒贬颠倒/重复赘余
```

#### 题型2：病句题
```
六大类型：
1. 语序不当（多重定语/状语顺序）
2. 搭配不当（主谓/动宾/主宾）
3. 成分残缺（缺主语/缺宾语）
4. 结构混乱（句式杂糅）
5. 表意不明（指代不明/歧义）
6. 不合逻辑（否定不当/自相矛盾）

检查口诀：主干找残缺、枝叶看搭配、语感通不通
```

#### 题型3：衔接题（语句复位）
```
解题步骤：
1. 看位置（段首→总起句、段中→过渡句/展开句、段尾→总结句）
2. 看关联（代词指代/关联词搭配/时间顺序/逻辑顺序）
3. 看标点（逗号/分号/句号提示关系层级）
4. 代入验证（读一遍是否通顺）
```
