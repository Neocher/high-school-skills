### 三、解答题标准答题模板（80+必会）

#### 模板1：三角函数与解三角形

```
已知：[条件]
求：[问题]

解：
1. 工具判断：[正弦定理/余弦定理/三角恒等变换]
2. 公式应用：[列出公式并代入]
3. 解方程：[求解过程]
4. 结论：[最终结果]

【踩分点】
├─ 公式写对（2分）
├─ 代入正确（2分）
├─ 计算正确（2分）
└─ 结论完整（2分）
```

#### 模板2：数列解答题

```
1. 判定数列类型 → 等差/等比（定义法或通项法）
2. 求通项公式 → 累加/累乘/待定系数/构造法
3. 求前n项和 → 公式法/裂项相消/错位相减/分组求和
4. 证明或讨论 → 单调性/最值/不等式放缩
```

#### 模板3：立体几何（向量法）

```
1. 建系 → 选原点，标坐标
2. 求方向向量/法向量
3. 代公式：线面角 sinθ=|cos<n,m>|，二面角 cosθ=cos<n₁,n₂>
4. 结论（注意线面角的正弦和余弦区别）
```

#### 模板4：概率统计

```
1. 确定概率模型（二项分布/超几何分布/正态分布）
2. 列出分布列（表格）
3. 求期望 E(X)=Σxᵢpᵢ
4. 根据期望做决策或比较方案
```

#### 模板5：解析几何

```
设直线方程（注意斜率不存在的情况）
联立方程组 → 韦达定理
利用几何条件（弦长/垂直/面积/定点等）列方程
化简求解（计算量通常较大，优先保证前两步得分）
```

#### 模板6：导数压轴（尽力得分策略）

```
1. 求导（定义域优先）
2. 讨论单调性（导函数零点分析）
3. 极值/最值计算
4. 不等式证明（构造函数/放缩）

80+策略：保证前两问满分（通常8分），第三问尽力
```

### 四、常见失分陷阱

| 陷阱 | 表现 | 应对 |
|:----|:-----|:-----|
| 定义域 | 对数真数>0、分母≠0、偶次根号≥0 | 写第一步先写定义域 |
| 判别式 | 二次方程/函数讨论时漏Δ | 分类讨论Δ>0/Δ=0/Δ<0 |
| 斜率不存在 | 设y=kx+b漏掉x=t的情况 | 先讨论斜率不存在 |
| 等比数列q=1 | 等比求和公式在q=1时分母为0 | 分q=1和q≠1讨论 |
| 函数单调性 | 导数正负判断错误 | 列表分析导数符号 |
| 概率模型混淆 | 二项分布vs超几何分布 | 放回→二项，不放回→超几何 |
| 向量法夹角 | 线面角sinθ≠|cosθ| 易错 | 线面角：sinθ=|cos<n,m>|
| 极值点偏移 | 构造对称函数的技巧 | 记住构造F(x)=f(x)-f(2x₀-x) |

### 五、核心公式速查

#### 函数与导数
- 导数公式：(xⁿ)'=nxⁿ⁻¹, (sinx)'=cosx, (cosx)'=-sinx, (eˣ)'=eˣ, (lnx)'=1/x
- 切线方程：y-f(x₀)=f'(x₀)(x-x₀)
- 极值判定：f'(x)=0且左正右负→极大值

#### 三角函数
- sin²x+cos²x=1, tanx=sinx/cosx
- 和差公式、倍角公式、辅助角公式
- 正弦定理 a/sinA=b/sinB=c/sinC=2R
- 余弦定理 a²=b²+c²-2bccosA

#### 数列
- 等差：an=a₁+(n-1)d, Sn=n(a₁+an)/2
- 等比：an=a₁qⁿ⁻¹, Sn=a₁(1-qⁿ)/(1-q)
- 裂项相消：1/n(n+1)=1/n-1/(n+1)

#### 解析几何
- 椭圆：x²/a²+y²/b²=1, c²=a²-b²
- 双曲线：x²/a²-y²/b²=1, c²=a²+b²
- 抛物线：y²=2px, 焦点(p/2,0), 准线x=-p/2
- 弦长公式：|AB|=√(1+k²)·|x₁-x₂|=√(1+k²)·√(Δ)/|A|
